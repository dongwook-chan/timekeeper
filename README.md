## Getting Started
```bash
# git clone
$ git clone {repo URL}
$ cd {repo name}
# create & activate venv
$ python3 -m venv env
$ source env/bin/activate
(env) $ python3 -m pip install -r requirements.txt
# activate pre-commit hooks
(env) $ pre-commit install
```
